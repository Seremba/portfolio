Event_booking App

The app for your event booking needs

The event booking app is an app built with HTML, CSS and Javascript. The backend is catered for by firebase and firestore database. where one fills information about the event and saves in the database. The app is authenticated in a way that it one cannot book an invent twice. If he tried, he will get a pop up message informing him that he has already booked the event. Events created show in home page!
