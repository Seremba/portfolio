RGB_Color Game

Fun game for colors

The color Game App is a fun game app that is developed with html, css and vanilla javascript. It has two parts. The hard and the easy part. The player must continue playing until he clicks on the right color. Then the whole page changes to the right color. Once that is done, one can reset then game and plays again
